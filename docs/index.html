Deepcut Enhancer

The ultimate toolkit for deepcut.live DJs and listeners.

Features

  • Auto-Voting: Automatically Awesome or Lame tracks. Smart delays to keep it looking natural.
  • Auto-Queue: Automatically hop on the decks when a spot opens up, or via chat mention.
  • Vote Tracking: See exactly who voted Awesome, Lame, or who hasn't voted yet.
  • Custom Themes & UI: Apply clean, dark, or customized CSS to deepcut.live.
  • AFK & Notifications: Set AFK messages and get desktop alerts when you're mentioned or when it's your turn to DJ.

Installation

Option 1: Chrome Extension (Recommended)

You can install this directly into Chrome as an unpacked extension for the best experience.

  1. Download or clone the repository.
  2. Go to chrome://extensions/ in your browser.
  3. Enable "Developer mode" in the top right.
  4. Click "Load unpacked" and select the deepcut-enhancer folder.

Option 2: Bookmarklet

Don't want an extension? Just drag this button to your bookmarks bar, and click it whenever you're in a deepcut.live room.

Drag to Bookmarks: Deepcut Enhancer
